## 4. 使用须知

### 4.1 演示环境限制
- **数据模拟**: 所有操作均在模拟环境中进行，不影响真实数据
- **配额限制**: 
  - AI调用次数: 5次/日
  - 内容发布次数: 10次/日
  - 数据导入次数: 3次/日
- **数据重置**: 所有演示数据每日自动重置
- **会话超时**: 演示账户会话24小时后自动过期

### 4.2 功能限制
- 某些高级功能可能被演示模式限制
- 付费功能在演示环境中以模拟方式呈现
- 实际API调用次数受限

### 4.3 数据保留策略
- 动态生成的演示数据每日清理
- 预置演示数据保持不变
- 用户操作历史定期清理

### 4.4 使用建议
- 建议先观看演示视频熟悉系统功能
- 遵循正常的业务流程进行体验
- 可随时重置演示数据获得全新体验
