## 1. 项目概述

LuminaMedia 3.0 DEMO版项目是一项为期半个多月的重要开发任务，旨在构建一个基于AI驱动的企业级内容营销平台。项目采用模块化单体架构，为企事业单位提供完整的宣传营销解决方案，支持SaaS多租户模式和私有化部署。

### 1.1 项目范围
- **技术架构**: NestJS + React + MySQL + Redis + Docker
- **AI集成**: Google Gemini + 阿里云通义千问
- **功能模块**: 22个模块化单体功能模块
- **部署模式**: SaaS多租户 + 私有化部署

## 5. 核心功能展示

### 5.1 功能开关系统
- **@Feature() 装饰器**: 控制器级别的功能开关
- **FeatureGuard**: 基于租户类型的功能访问控制
- **FeatureConfigService**: 功能配置CRUD服务
- **TenantFeatureService**: 租户功能开关服务

### 5.2 配额限制系统
- **QuotaService**: 配额配置和使用统计
- **QuotaCheckMiddleware**: API层配额检查中间件
- **每日配额限制**: AI调用/内容发布/数据导入限制
- **管理员重置**: 配额手动重置功能
