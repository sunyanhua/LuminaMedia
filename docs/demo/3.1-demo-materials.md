## 3. 录屏准备

### 3.1 核心流程录屏
**录屏1: 完整演示流程** (时长: 15-20分钟)
- **内容**: 从登录到总结的完整流程
- **用途**: 备用演示、培训材料、产品介绍
- **格式**: MP4, 1080p, 30fps
- **音频**: 包含讲解配音或字幕
- **存储**: `docs/demo/videos/full-demo.mp4`

**录屏2: AI内容创作流程** (时长: 3分钟)
- **内容**: 从空白到AI生成完整内容的流程
- **用途**: 展示AI能力，社交媒体分享
- **格式**: MP4, 1080p, 30fps
- **音频**: 轻背景音乐，关键步骤字幕
- **存储**: `docs/demo/videos/ai-creation.mp4`

**录屏3: 三审三校工作流** (时长: 4分钟)
- **内容**: 多角色切换完成审批的全过程
- **用途**: 展示工作流，内部培训
- **格式**: MP4, 1080p, 30fps
- **音频**: 步骤说明配音
- **存储**: `docs/demo/videos/review-workflow.mp4`

### 3.2 录屏制作要求
1. **软件**: 使用专业录屏软件（OBS、Camtasia等）
2. **帧率**: 不低于30fps
3. **码率**: 不低于5000kbps
4. **音频**: 清晰无杂音，音量均衡
5. **字幕**: 关键步骤添加字幕说明
6. **片头片尾**: 简单的产品标识和联系方式
7. **长度控制**: 严格控制在计划时间内

### 3.3 录屏脚本
**完整演示录屏脚本** (参考):
```
00:00-00:30 片头动画和平台介绍
00:30-01:30 登录和工作台展示
01:30-03:30 智慧档案和单位画像
03:30-05:30 参考信息和一键采用
05:30-10:30 AI内容创作完整流程
10:30-13:30 三审三校工作流
13:30-16:30 舆情监测和数据看板
16:30-18:30 一键报告生成和导出
18:30-20:00 总结和片尾
```
