# 商务版DEMO使用指南

## 概述

商务版DEMO是LuminaMedia内容营销平台的演示版本，专门为企业客户设计。它展示了平台如何帮助企业进行数据驱动的营销决策、AI智能内容生成和多渠道内容发布。

## 目标受众

- **企业营销负责人**: 了解平台如何提升营销效率
- **数据分析师**: 查看数据分析和洞察功能
- **内容创作者**: 体验AI辅助内容生成
- **技术决策者**: 评估平台技术能力和集成方案

## 核心功能演示

### 1. 数据导入与分析

**演示目标**: 展示如何导入客户数据并获得有价值的商业洞察

**操作步骤**:
1. 登录系统后进入**仪表板**页面
2. 点击左侧菜单的**数据引擎**
3. 点击"导入数据"按钮，选择示例数据文件 (`demo/sample-data/customers.csv`)
4. 等待数据导入完成（约2-3秒模拟时间）
5. 查看数据分析结果，包括：
   - 客户分布分析
   - 消费行为模式
   - 用户细分群体

**关键看点**:
- 实时数据可视化图表
- AI生成的洞察摘要
- 可交互的数据探索界面

### 2. AI营销策略生成

**演示目标**: 展示AI如何基于数据分析生成营销策略

**操作步骤**:
1. 在**AI工作流**模块中点击"新建营销策略"
2. 选择目标客户细分（如"高价值客户"）
3. 设置营销目标（如"提升复购率"）
4. 选择内容类型（如"促销活动"）
5. 点击"生成策略"按钮
6. 查看AI生成的完整营销方案，包括：
   - 目标受众分析
   - 渠道选择建议
   - 内容创意方向
   - 预算分配建议
   - ROI预测

**关键看点**:
- AI生成的个性化策略
- 基于数据的决策支持
- 多方案对比功能

### 3. 内容生成与优化

**演示目标**: 展示AI辅助内容生成和优化能力

**操作步骤**:
1. 在**内容工场**模块中选择"创建内容"
2. 选择内容类型（如"微信公众号文章"）
3. 输入主题（如"五一促销活动"）
4. 选择风格（如"正式"、"亲切"、"激励"）
5. 点击"生成内容"按钮
6. 查看AI生成的内容草稿，可进行：
   - 内容编辑和优化
   - 关键词调整
   - 语气风格修改
   - A/B测试版本创建

**关键看点**:
- 多平台内容适配
- 实时内容优化建议
- 品牌一致性检查

### 4. 矩阵发布与监控

**演示目标**: 展示多渠道发布和效果监控

**操作步骤**:
1. 在**矩阵发布**模块中选择"新建发布任务"
2. 选择要发布的内容
3. 选择目标平台（如微信公众号、小红书、微博）
4. 设置发布时间（立即发布或定时发布）
5. 点击"发布"按钮
6. 在**监控中心**查看发布效果，包括：
   - 各平台发布状态
   - 实时互动数据
   - 效果分析报告
   - ROI计算

**关键看点**:
- 一站式多平台发布
- 实时效果监控
- 自动化效果报告

### 常见问题与回答

**Q: 真实数据导入需要多长时间？**
A: 实际使用中，10万条客户数据导入和分析约需5-10分钟，具体取决于数据复杂度和服务器性能。

**Q: AI生成的内容质量如何？**
A: 平台使用先进的AI模型，生成内容经过优化和审核，质量接近专业营销人员水平。用户可随时进行人工调整。

**Q: 支持哪些社交媒体平台？**
A: 目前支持微信公众号、小红书、微博、抖音等主流平台，更多平台持续增加中。

**Q: 如何保证数据安全？**
A: 采用企业级数据加密、多租户隔离、访问控制等多种安全措施，符合GDPR等数据保护法规。

## 故障排除

### 常见问题

**1. 无法访问DEMO**
- 检查Docker服务是否运行: `docker ps`
- 检查端口是否被占用: `netstat -ano | findstr :5175`
- 查看服务日志: `docker-compose -f docker-compose.demo.yml logs demo-frontend`

**2. 数据导入失败**
- 检查数据文件格式是否符合要求
- 查看导入日志: `docker-compose -f docker-compose.demo.yml logs demo-business`
- 重置DEMO数据: `docker-compose -f docker-compose.demo.yml exec demo-business npm run demo:reset`

**3. AI功能异常**
- 检查AI服务API密钥配置
- 验证网络连接
- 查看AI服务日志
