# 3.1 DEMO版 Phase 5: 舆情监测 + 一键报告

**阶段目标**: 实现舆情监测面板、智能报告生成、自定义报告功能  
**预计周期**: 4天  
**开始时间**: 待定  
**依赖阶段**: Phase 4 三审三校

---

## 任务清单

### Day 21-22: 舆情监测

#### 21.1 舆情数据实体
- [ ] 创建舆情监测实体
  - 字段：id, tenantId, platform, content, author, publishTime, sentiment, sentimentScore, readCount, shareCount, commentCount, keywords, url
  - 平台：WEIBO(微博), WECHAT(微信), DOUYIN(抖音), XIAOHONGSHU(小红书)
  - 情感：POSITIVE(正面), NEGATIVE(负面), NEUTRAL(中性)
- [ ] 舆情数据初始化（DEMO数据）
  - 插入模拟舆情数据
  - 覆盖不同平台和情感类型
  - 包含与政务相关的热点话题

#### 21.2 后端API
- [ ] 舆情列表查询API
  - 分页查询
  - 按平台筛选
  - 按情感筛选
  - 按关键词搜索
  - 按时间范围筛选
  - 排序（时间、热度）
- [ ] 舆情统计API
  - 今日舆情总数
  - 情感分布统计
  - 平台分布统计
  - 热点关键词
- [ ] 舆情详情API
  - 单条舆情完整信息
  - 相关评论列表
  - 传播路径（模拟）

#### 21.3 舆情监测页面
- [ ] 舆情总览区域
  - 今日新增舆情数
  - 情感分布饼图
  - 平台分布柱状图
  - 热点关键词词云
- [ ] 舆情列表
  - 表格形式展示
  - 列：平台图标、内容摘要、情感标签、热度指标、时间
  - 情感标签颜色区分（绿色正面、红色负面、灰色中性）
  - 关键词高亮显示
  - 操作：【查看详情】
- [ ] 筛选功能
  - 平台筛选器（多选）
  - 情感筛选器（多选）
  - 时间范围选择
  - 关键词搜索框
  - 重置筛选按钮

#### 22.1 舆情详情
- [ ] 详情弹窗/页面
  - 完整舆情内容
  - 作者和发布时间
  - 来源平台链接（可跳转）
  - 情感分析详细说明
  - 相关评论列表
  - 传播路径图（简单的传播链路展示）
- [ ] 舆情预警设置（可选）
  - 预警阈值设置
  - 负面舆情达到阈值时标记为预警

**交付物**:
- 舆情监测实体和API
- 舆情监测总览页面
- 舆情列表和详情功能

### Day 23: 智能报告

#### 23.1 报告模板设计
- [ ] 定义报告类型
  - 舆情监测日报
  - 舆情监测周报
  - 公众号运营月报
  - 传播分析报告
- [ ] 报告数据结构
  - 报告元数据（标题、生成时间、租户信息）
  - 报告内容（各章节数据）
  - 图表数据

#### 23.2 后端报告生成服务
- [ ] 舆情监测报告生成
  - 汇总指定时间段舆情数据
  - 情感分析统计
  - 热点话题总结
  - AI生成分析结论和建议
- [ ] 公众号运营报告生成
  - 粉丝增长分析
  - 内容发布统计
  - 阅读/点赞/转发数据分析
  - AI生成运营建议
- [ ] 报告生成API
  - 选择报告类型和时间范围
  - 触发AI生成报告
  - 保存报告到数据库
  - 返回报告ID

#### 23.3 智能报告页面
- [ ] 报告生成入口
  - 选择报告类型（下拉框）
  - 选择时间范围（日期选择器）
  - 生成按钮
  - 生成进度显示
- [ ] 报告列表
  - 显示已生成的报告
  - 列：报告类型、时间范围、生成时间、操作
  - 操作：【查看】【导出】
- [ ] 报告详情页面
  - 报告标题和基本信息
  - 各章节内容展示
  - 数据图表嵌入
  - 分析结论和建议
  - 导出按钮

**交付物**:
- 报告生成服务
- 智能报告生成和查看功能

### Day 24: 自定义报告

#### 24.1 用户文档存储
- [ ] 创建用户文档实体（独立于租户知识库）
  - 字段：id, userId, title, fileUrl, fileType, description, createdAt
  - 说明：用户上传的用于生成报告的参考文档，存储于用户个人空间
- [ ] 用户文档上传API
  - 支持Word/PDF上传
  - 关联到当前用户
  - 提取文档内容

#### 24.2 自定义报告生成服务
- [ ] 后端自定义报告生成
  - 接收用户上传的文档
  - AI分析文档内容提取关键信息
  - 基于文档内容生成报告结构
  - 生成完整报告内容
  - 支持报告类型：工作总结、活动复盘、调研分析、政策解读等
- [ ] 报告生成API
  - 上传文档并指定报告类型
  - AI生成报告
  - 返回报告内容

#### 24.3 自定义报告页面
- [ ] 文档上传区域
  - 选择报告类型（工作总结/活动复盘/调研分析/其他）
  - 文档上传（支持拖拽）
  - 文档描述输入
  - 开始生成按钮
- [ ] 生成结果编辑
  - 报告内容编辑器
  - 支持章节调整
  - 支持内容修改
  - 保存按钮
- [ ] 导出功能
  - 导出为Word格式
    - 后端生成Word文档
    - 提供下载链接
  - 生成PPT大纲
    - AI生成PPT结构大纲
    - 显示每页PPT标题和要点
    - 可复制到PPT软件使用

#### 24.4 Word导出服务
- [ ] 后端Word生成
  - 使用docx库生成Word文档
  - 支持标题、段落、图片、表格
  - 应用政务报告样式模板
  - 提供下载接口

**交付物**:
- 用户文档存储功能
- 自定义报告生成（基于上传文档）
- Word导出功能
- PPT大纲生成功能
